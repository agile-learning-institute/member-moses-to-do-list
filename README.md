## Setup Locally
1. Fork and Clone the repository  
    `git clone git@github.com:agile-learning-institute/member-moses-to-do-list.git`  

2. Install Packages  
    `cd member-moses-restaurant-page`  
    `npm install`  

3. Spin it up  
    `npm run build`  
    `npm run start`
